About This Tool

This platform performs comprehensive network performance analysis for cellular networks, processing data from multiple vendors. It forecasts congestion, identifies bottlenecks, and provides actionable insights.

Supported Formats

  • • xC Format (Celcom): Daily data with Date column.
  • • xD Format (Digi): Weekly aggregated data.
  • • CSV, Excel (XLS, XLSX, XLSB)

Key Features

  • • Dual Pipeline: Adaptive logic for xC vs xD.
  • • 52-week Forecasting: AI-driven prediction.
  • • Congestion Alerts: Multi-metric thresholds.
  • • CD File Export: Unified granular data download.

Naming Logic

  • • Celcom: Identified by "_" (e.g. ..._7_1 -> Sector 1)
  • • Digi: Identified by "-" (e.g. ...-21 -> Sector 1)
  • • Automatic Operator Detection
